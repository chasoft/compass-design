# File

The File component allows you to display state and preview of the file uploaded.

---

### Import:

```js
import {File} from '@comfortdelgro/react-compass'
```

or

```js
import File from '@comfortdelgro/react-compass/file'
```

---

### Basic State:

<section className={styles.avatarShowcaseContainer} style={{ flexDirection: 'column' }}>
  <File.State fileName='Filename.png' icon='close' />
  <File.State fileName='Filename.png' icon='error' />
  <File.State fileName='Filename.png' icon='success' />
  <File.State fileName='Filename.png' icon='download' />
  <File.State fileName='Filename.png' icon='loading' />
</section>

### Custom styling:

You can style React Compass's components with the css prop. <br/>
With css prop, you can basically do anything that css can do.

<section className={styles.avatarShowcaseContainer}>
<File.State 
  fileName='Filename.png' 
  icon='close' 
  imageIcon={<Icon icon={faVirus} />} 
  css={{
    background: 'red',
  }}
/>
</section>

```js
<File.State 
  fileName='Filename.png' 
  icon='close' 
  imageIcon={<Icon icon={faVirus} />} 
  css={{
    background: 'red',
  }}
/>
```

### File State Props:
| Name        | Type                                               | Default                | Description                                                                             |
| :-----------| :--------------------------------------------------| :----------------------|:----------------------------------------------------------------------------------------|
| fileName    | `string`                                           | -                      | Filename of the state                                                                   |
| imageIcon   | `React.ReactNode`                                  | -                      | Little icon display on left of the filename                                             |
| onIconPress | `() => void`                                       | -                      | Fired when clicked on the icon                                                          |
| icon        | `close`\|`error`\|`success`\|`download`\|`loading` | Left icon of the state |                                                                                         |
| css         | `CSS`                                              | —                      | The system prop that allows defining system overrides as well as additional CSS styles. |

### File Preview Props:
| Name        | Type                                               | Default                | Description                                                                             |
| :-----------| :--------------------------------------------------| :----------------------|:----------------------------------------------------------------------------------------|
| fileName    | `string`                                           | -                      | Filename of the state                                                                   |
| imageSrc    | `string`                                           | -                      | Image source to display                                                                 |
| imageIcon   | `React.ReactNode`                                  | -                      | Little icon display on left of the filename                                             |
| onIconPress | `() => void`                                       | -                      | Fired when clicked on the icon                                                          |
| icon        | `close`\|`error`\|`success`\|`download`\|`loading` | Left icon of the state |                                                                                         |
| css         | `CSS`                                              | —                      | The system prop that allows defining system overrides as well as additional CSS styles. |
